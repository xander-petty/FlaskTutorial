# Flask Tutorial

## Description

Introduction tutorial to Flask

## Installation

```bash
python3 -m venv .
source ./bin/activate
pip install --upgrade pip setuptools
pip install -r requirements.txt
```

## Usage

```bash
flask --app app run --debug
```
OR
```bash
flask --app app run --host 0.0.0.0 --port 80
```
